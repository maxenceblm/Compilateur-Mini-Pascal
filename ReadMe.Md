# Projet Compilateur pour un langage Mini Pascal en Python

# Compilation et Exécution :

Langage requis : Python 3.10+ 
Aucune dépendance nécessaire.
----------------------------------
# Exécution :

python compilateur.py

Il faut changer le nom du fichier qu'on
veut tester dans la méthode main de compilateur.py

----------------------------------

# Résultat :

- Affiche la table des Identificateurs 
- Génère un fichier nomfichier.cod contenant le P-Code lisible 
- Exécute le programme
